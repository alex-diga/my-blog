# 类型判断

## typeof

```js
function getType(obj) {
  return obj === null ? null : typeof obj;
}
```

## Object.prototype.toString

```js
function getType(obj) {
  const type = Object.prototype.toString();
  const regE = /^\[object (\w*)\]$/g;
  return type.replace(regE, (_, val) => val);
}
```
